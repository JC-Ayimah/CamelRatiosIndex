# Changelog

## CamelRatiosIndex (development version)

### CamelRatiosIndex 0.1.0

- Initial CRAN release.
- Added
  [`camel_index()`](https://JC-Ayimah.github.io/CamelRatiosIndex/reference/camel_index.md)
  for computing multivariate-weighted CAMEL indices.
- Added
  [`plot_camel_index()`](https://JC-Ayimah.github.io/CamelRatiosIndex/reference/plot_camel_index.md)
  for ggplot2-based visualization.
- Added
  [`print.camel_index()`](https://JC-Ayimah.github.io/CamelRatiosIndex/reference/print.camel_index.md)
  and
  [`summary.camel_index()`](https://JC-Ayimah.github.io/CamelRatiosIndex/reference/summary.camel_index.md)
  methods.
- Added built-in datasets `camel_2015`, `camel_2016`, `camel_2017`,
  `camel_2018`, `camel_2019`, `camel_2020`, `camel_2021` and
  `camel_2022`.
- Set up CI/CD with GitHub Actions.
- Added package website with pkgdown.
