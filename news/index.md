- Initial CRAN release.

  [`camel_index()`](https://YOUR-USERNAME.github.io/CamelRatiosIndex/reference/camel_index.md)
  for computing multivariate-weighted CAMEL indices.
- Added
  [`plot_camel_index()`](https://YOUR-USERNAME.github.io/CamelRatiosIndex/reference/plot_camel_index.md)
  for ggplot2-based visualization.
- Added
  [`print.camel_index()`](https://YOUR-USERNAME.github.io/CamelRatiosIndex/reference/print.camel_index.md)
  and
  [`summary.camel_index()`](https://YOUR-USERNAME.github.io/CamelRatiosIndex/reference/summary.camel_index.md)
  methods.
- Added built-in datasets `camel_2015` and `camel_2022`.
- Set up CI/CD with GitHub Actions.
- Added package website with pkgdown.
